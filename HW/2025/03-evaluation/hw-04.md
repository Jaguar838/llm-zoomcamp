Також нам знадобиться код для оцінювання якості пошуку:

```python
from tqdm.auto import tqdm

def hit_rate(relevance_total):
    cnt = 0
    for line in relevance_total:
        if True in line:
            cnt += 1
    return cnt / len(relevance_total)

def mrr(relevance_total):
    total_score = 0.0
    for line in relevance_total:
        for rank in range(len(line)):
            if line[rank] == True:
                total_score += 1 / (rank + 1)
    return total_score / len(relevance_total)

def evaluate(ground_truth, search_function):
    relevance_total = []
    for q in tqdm(ground_truth):
        doc_id = q['document']
        results = search_function(q)
        relevance = [d['id'] == doc_id for d in results]
        relevance_total.append(relevance)

    return {
        'hit_rate': hit_rate(relevance_total),
        'mrr': mrr(relevance_total),
    }
```

## Питання 5. Косинусна подібність (Cosine similarity)

У другій частині модуля ми порівнювали відповіді системи з оригінальними відповідями з FAQ.

Один зі способів зробити це — обчислити косинусну подібність:

```python
def cosine(u, v):
    u = normalize(u)
    v = normalize(v)
    return u.dot(v)

def normalize(u):
    norm = np.sqrt(u.dot(u))
    return u / norm
```

Або простіше:

```python
def cosine(u, v):
    u_norm = np.sqrt(u.dot(u))
    v_norm = np.sqrt(v.dot(v))
    return u.dot(v) / (u_norm * v_norm)
```

Завантажимо дані результатів:


Вона базується на збігах між n-грамами, послідовностями слів та підрядками.

Встановіть бібліотеку:


(Остання версія — `1.0.1`)

Порахуємо ROUGE для документа з індексом 10:
