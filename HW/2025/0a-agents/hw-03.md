## Підготовка

Спочатку ми визначимо функцію, яку будемо використовувати при створенні нашого агента.
Вона буде **генерувати фейкові погодні дані**:

```python
import random

known_weather_data = {
    'berlin': 20.0
}

def get_weather(city: str) -> float:
    city = city.strip().lower()

    if city in known_weather_data:
        return known_weather_data[city]

    return round(random.uniform(-5, 35), 1)
```

## MCP

**MCP** означає _Model-Context Protocol_. Це дозволяє LLM'ам взаємодіяти з інструментами (наприклад, Qdrant). Це **функціональний виклик, але на рівень вище**:

- Інструмент може надавати список функцій, які він підтримує
- Додаючи інструмент до агента, достатньо вказати посилання на MCP-сервер

## Питання 4. Простий MCP-сервер

Простий приклад MCP-сервера виглядає так:

```python
# weather_server.py
from fastmcp import FastMCP

mcp = FastMCP("Demo 🚀")

@mcp.tool
def add(a: int, b: int) -> int:
    """Add two numbers"""
    return a + b

if __name__ == "__main__":
    mcp.run()
```

Для нас потрібно написати **docstring-и** для наших функцій. Ось приклад:

```python
def get_weather(city: str) -> float:
    """
    Отримує температуру для заданого міста.

    Параметри:
        city (str): Назва міста, для якого потрібно отримати дані про погоду.

    Повертає:
        float: Температура у місті.
    """

def set_weather(city: str, temp: float) -> None:
    """
    Встановлює температуру для заданого міста.

    Параметри:
        city (str): Назва міста.
        temp (float): Температура, яку потрібно зберегти.

    Повертає:
        str: Підтвердження оновлення ('OK').
    """
```

## Питання 5. Протокол

Наш сервер працює через **stdin/stdout**.

Порядок взаємодії:

1. Ініціалізація клієнта:

   ```json
   {"jsonrpc": "2.0", "id": 1, "method": "initialize", ...}
   ```

2. Підтвердження:

   ```json
   { "jsonrpc": "2.0", "method": "notifications/initialized" }
   ```

3. Отримати список методів:

   ```json
   { "jsonrpc": "2.0", "id": 2, "method": "tools/list" }
   ```

4. Виклик функції, наприклад, погода в Берліні:

   ```json
   {
     "jsonrpc": "2.0",
     "id": 3,
     "method": "tools/call",
     "params": { "name": "<TODO>", "arguments": { "city": "Berlin" } }
   }
   ```

## Додатково: простий клієнт MCP (необов’язково)

Є також **не-асинхронний клієнт** `mcp_client.py` (спрощений). Його можна використати так:

```python
import mcp_client

our_mcp_client = mcp_client.MCPClient(["python", "weather_server.py"])

our_mcp_client.start_server()
our_mcp_client.initialize()
our_mcp_client.initialized()

our_mcp_client.get_tools()
our_mcp_client.call_tool('get_weather', {'city': 'Berlin'})
```

Для інтеграції в застосунок — використовуйте обгортку:

```python
class MCPTools:
    def __init__(self, mcp_client):
        self.mcp_client = mcp_client
        ...
```

Використання:

```python
chat = chat_assistant.ChatAssistant(
    tools=mcp_tools,
    developer_prompt=developer_prompt,
    ...
)

chat.run()
```
